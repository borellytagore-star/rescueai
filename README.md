# 🚑 RescueAI – AI-Powered Emergency Response Assistant

RescueAI is an intelligent emergency response web application that helps users react quickly during medical and safety emergencies. Using AI-powered emergency analysis, voice input, image-based assessment, live location services, and one-tap SOS alerts, RescueAI provides immediate guidance before professional responders arrive.

---

## 🌟 Features

- 🧠 **AI Emergency Analysis**
  - Detects emergency type
  - Estimates severity
  - Suggests immediate actions

- 🎤 **Voice-Based Emergency Input**
  - Hands-free emergency reporting
  - Real-time speech recognition

- 📷 **Image Analysis**
  - Upload emergency images
  - AI-assisted visual assessment

- 📍 **Nearby Hospitals**
  - Find nearby hospitals and emergency services
  - Location-based recommendations

- 🚨 **One-Tap SOS**
  - Share emergency alerts instantly
  - Send current location to emergency contacts

- ❤️ **First Aid Guidance**
  - Step-by-step emergency instructions
  - Offline accessible emergency guides

- 📊 **Emergency History**
  - Track previous emergency reports
  - Review AI recommendations

- ⚙️ **Settings Management**
  - Customize emergency preferences
  - Manage contacts and notifications

# 🚀 Getting Started

## Clone Repository

```bash
git clone https://github.com/yourusername/rescueai.git
```

```bash
cd rescueai
```

## Install Dependencies

```bash
npm install
```

## Run Development Server

```bash
npm run dev
```

Open:

```
http://localhost:5173
```

---

# 📦 Build for Production

```bash
npm run build
```

Preview production build:

```bash

```
